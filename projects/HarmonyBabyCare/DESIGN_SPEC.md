## 📐 组件规范

### 快速记录按钮

```typescript
// 圆形快速操作按钮
- 尺寸: 64vp × 64vp
- 圆角: 32vp (完全圆形)
- 图标: 28vp
- 背景: 模块对应色 (10% 透明度)
- 激活: 模块对应色 (100%)
```

### 计时器显示

```typescript
// 大数字计时器
- 字号: 56fp
- 字重: Bold (700)
- 颜色: text_primary
- 字体: Monospace (等宽)
- 格式: HH:MM:SS 或 MM:SS
```

### 数据卡片

```typescript
// 数据统计卡片
- 内边距: 16vp
- 圆角: 16vp
- 背景: bg_primary
- 阴影: 0 2vp 8vp rgba(0,0,0,0.06)
- 标题: 14fp, medium, text_secondary
- 数值: 24fp, bold, text_primary
- 单位: 12fp, regular, text_tertiary
```

## 🎬 动效规范

### 时长

| 场景 | 时长 | 曲线 |
|------|------|------|
| 按钮反馈 | 100ms | EaseOut |
| 卡片展开 | 250ms | EaseInOut |
| 页面切换 | 300ms | FastOutSlowIn |
| 计时器闪烁 | 500ms | Linear |

### 关键动效

1. **记录成功** - 卡片轻微弹跳 + 对勾动画
2. **计时进行中** - 秒钟闪烁 + 进度环动画
3. **语音激活** - 波形脉冲动画
4. **快速记录** - 图标缩放 + 涟漪效果

## ♿ 无障碍设计

1. **对比度** - 所有文字对比度 ≥ 4.5:1
2. **触摸目标** - 最小 44vp × 44vp
3. **语音描述** - 所有按钮提供 accessibilityText
4. **焦点顺序** - 逻辑阅读顺序
5. **动效减弱** - 支持系统减弱动效设置
